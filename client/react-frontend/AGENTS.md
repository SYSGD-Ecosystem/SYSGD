# Reglas para la IA

## Metodo de coneccion con el servidor

- El proyecto usa una api en la carpeta lib, todas las peticiones se realisaran utilisando la api en la libreria, nunca debe hacerse feth independiente

## Reglas de programacion general

- Nunca usar any, siempre respetar las buenas practicas de typescript

- Todao los componentes deben ser responsibos

- Todos los elementos de UI tendran modo claro, modo ocuro.
